# Chapter 03: Introducing Lists

## 📝 Overview
This folder contains my progress through Chapter 3 of *Python Crash Course*. The exercises focus on mastering Python lists—a critical skill for data management and DevSecOps automation.

## 🚀 Skills Demonstrated
* **Dynamic Modification:** Adding, removing, and replacing elements in real-time.
* **Sorting Logic:** Using `sort()` and `reverse()` for permanent changes and `sorted()` for temporary data views.
* **Memory Management:** Using `pop()` to retrieve and remove items, and `del` for permanent deletion.
* **Error Handling:** Troubleshooting `IndexError` and understanding list indices (starting at 0).

## 📂 Exercise Highlights
* **Guest List, Shrinking Guest List, Dinner Guests:** Managed a dynamic invitation list, handling changes in guest availability and table capacity.
* **Seeing the World:** Practiced organizing data without losing the original state of the information.
* **Every Function:** A capstone script implementing all chapter methods on a list of languages.
* **Intentional Error:** Intentionally triggered and resolved an `IndexError` to practice reading Python tracebacks.
