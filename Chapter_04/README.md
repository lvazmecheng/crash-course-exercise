# Chapter 04: Working with Lists

## 📝 Overview
This folder contains my progress through Chapter 4 of *Python Crash Course*. These exercises focus on mastering list iteration, data segmentation, and the introduction of immutable data structures—essential building blocks for writing efficient automation scripts.

## 🚀 Skills Demonstrated
* **Efficient Iteration:** Using `for` loops to automate repetitive tasks across entire datasets.
* **Data Generation:** Utilizing `range()` and List Comprehensions to create complex numerical lists with minimal code.
* **Data Segmentation:** Mastering slicing techniques (`[start:stop]`) and negative indexing to isolate specific data chunks.
* **Immutability:** Implementing Tuples for "read-only" data to ensure system constants remain untampered.
* **Clean Code Standards:** Applying PEP 8 styling rules

## 📂 Exercise Highlights
* **Pizzas & Animals:** Practiced basic iteration and the importance of loop indentation.
* **Cubes & Cube Comprehension:** Compared traditional loop-based data generation with the performance and brevity of list comprehensions.
* **Slices:** Demonstrated the ability to extract the first, middle, and last segments of a list.
* **My Pizzas, Your Pizzas:** Used list slicing (`[:]`) to create independent deep copies of data, preventing accidental modification of original sources.
* **Buffet:** Built an immutable menu using Tuples and practiced variable re-assignment to "update" locked datasets.
