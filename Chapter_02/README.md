# Chapter 2: Variables and Simple Data Types
---

## 📝 Overview
This chapter covers the absolute building blocks of Python. I practiced managing data using variables and working with different types of data, specifically **Strings**, **Numbers**, and **Comments**.

## 💡 Key Concepts Practiced
- **Variables**: Assigning and changing values, and following Python naming conventions.
- **Strings**: Using methods like `.title()`, `.upper()`, and `.lower()`.
- **f-strings**: Combining variables with strings for dynamic output.
- **Whitespaces**: Managing layouts using `\t` (tabs) and `\n` (newlines).
- **Numbers**: Handling integers, floats, and the use of underscores in large numbers (e.g., `1_000_000`).

- `famous_quote.py`: Using variables to store and print formatted text.
- `famous_quote2.py`: Using more than one variable with f string to print message.

- `file_extensions.py`:removing the extension from a filename in string format using the removesuffix method.
- `number_eight.py`: With basic math operators in python used addition, substraction, multiplication and division.
- `favorite_number.py`: Using f string to take a variable with an integer value and print it in a string message.
- `adding_comments.py`: Demonstrating the use of Hash or pound sign to generate comments.
- `zen_of_python.py`: Used import this which display the guiding principles for writing readable and maintainable code.
