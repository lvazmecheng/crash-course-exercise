# crash-course-exercise
Solutions and exercises from 'Python Crash Course' by Eric Matthes to demonstrate Python fundamentals and project-based learning.

This repository contains my personal solutions to the exercises found in **"Python Crash Course: A Hands-On, Project-Based Introduction to Programming"** by **Eric Matthes** (published by [No Starch Press](https://nostarch.com)).

## 📋 About This Project
The purpose of this repository is to showcase my progress through the book and document my learning journey as I master Python fundamentals. Each folder corresponds to a chapter in the book, containing the "Try It Yourself" exercises.

## 🚀 Key Learning Objectives
Through these exercises, I am developing skills in:
- **Python Fundamentals**: Variables, lists, dictionaries, and loops.
- **Functional Programming**: Writing clean, reusable functions.
- **Object-Oriented Programming (OOP)**: Creating and managing classes.
- **File Handling & Testing**: Working with external data and writing unit tests with `pytest`.
